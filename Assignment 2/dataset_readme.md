This dataset contains listings of cars and trucks collected from Craigslist, a popular online marketplace where individuals and dealerships sell vehicles. The data includes information about vehicle specifications, pricing, condition, location, and listing details.

The dataset is useful for data analysis, price prediction, and machine learning tasks such as regression, classification, and exploratory data analysis.

Dataset Contents

# The dataset contains vehicle listings with attributes such as

price – Listing price of the vehicle

year – Manufacturing year

manufacturer – Vehicle brand

model – Specific vehicle model

condition – Vehicle condition (e.g., excellent, good, fair)

cylinders – Engine cylinder count

fuel – Fuel type (gas, diesel, electric, etc.)

odometer – Mileage of the vehicle

transmission – Transmission type (automatic, manual)

drive – Drive type (FWD, RWD, 4WD)

type – Vehicle type (sedan, SUV, truck, etc.)

paint_color – Vehicle color

state / region – Location of the listing
